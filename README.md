# Electric-Vehicle-Data-Analysis-Dashboard
🚗⚡ Electric Vehicle Data Analysis Dashboard | Tableau Project I’m excited to share my latest Electric Vehicle (EV) Data Analysis Dashboard built using Tableau, where I analyzed EV adoption patterns across the United States using interactive data visualizations.

🔧 What I Did in This Project
• Cleaned and prepared EV registration data for analysis
• Built an interactive dashboard in Tableau with filters for CAFV Eligibility, EV Type, Model, Model Year, and State
• Designed KPI cards to track key metrics like Total Vehicles, BEV vs PHEV distribution, and Average Electric Range
• Created visualizations including state-wise EV distribution map, EV adoption trend by model year, top manufacturers, and model-wise analysis
• Enabled dynamic filtering to allow deeper exploration of EV trends across different regions and brands
📊 Key Insights from the Dashboard
• Total vehicles analyzed: 150,413
• Battery Electric Vehicles (BEV) dominate the market with 77.62% share, while Plug-in Hybrid Electric Vehicles (PHEV) account for 22.4%
• Tesla leads the EV market with more than 51% of total vehicles, followed by Nissan and Chevrolet
• Model Y and Model 3 are the most popular EV models in the dataset
• EV adoption has grown rapidly after 2020, reaching peak registrations around 2023
• State-level analysis highlights regional differences in EV adoption across the U.S.
• CAFV eligibility analysis shows how many vehicles qualify for clean fuel incentive programs
📈 This project helped me strengthen my skills in data visualization, dashboard design, and data storytelling, turning raw EV data into meaningful insights.
